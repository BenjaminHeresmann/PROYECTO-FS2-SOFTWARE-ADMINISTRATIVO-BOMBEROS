# 2DA COMPAÑÍA BOMBEROS - Sistema Web 🚒

## Descripción
Este es un proyecto web estudiantil que simula el sistema de una compañía de bomberos, creado como parte del aprendizaje en HTML, CSS y JavaScript vanilla. Es una versión educativa y simplificada basada en mockups de diseño.

## Temática del Proyecto
- **Organización**: 2DA COMPAÑÍA DE BOMBEROS
- **Propósito**: Sistema web informativo y de gestión básica
- **Enfoque**: Educativo para aprender desarrollo frontend

## Tecnologías Utilizadas
- **HTML5**: Para la estructura básica de la página
- **CSS3**: Para el diseño y estilos visuales siguiendo el mockup
- **JavaScript**: Para la interactividad básica

## Características del Proyecto
- ✅ Diseño basado en mockup profesional
- ✅ Temática de bomberos con contenido coherente
- ✅ Grid de bomberos con tarjetas interactivas
- ✅ Formulario de solicitud de ingreso funcional
- ✅ Sistema de citaciones informativo
- ✅ Navegación suave entre secciones
- ✅ Diseño responsive para móviles y tablets
- ✅ Interacciones educativas con JavaScript vanilla

## Conceptos Aprendidos

### HTML
- Estructura básica de documento HTML
- Elementos semánticos (header, nav, main, section, footer)
- Formularios y elementos de entrada
- Enlaces internos y navegación
- Atributos básicos (id, class, href, etc.)

### CSS
- Selectores básicos y especificidad
- Propiedades de texto y color
- Modelo de caja (margin, padding, border)
- Flexbox y Grid básico
- Media queries para responsive design
- Transiciones y efectos hover
- Posicionamiento (fixed, relative)

### JavaScript
- Selección de elementos del DOM
- Event listeners (click, submit, scroll)
- Funciones básicas y variables
- Condicionales y bucles
- Validación de formularios
- Manipulación básica del DOM
- SetInterval y setTimeout
- Console.log para debugging

## Cómo Usar el Proyecto
1. Descarga todos los archivos en la misma carpeta
2. Abre `index.html` en tu navegador web
3. Explora las diferentes secciones navegando por el menú
4. Prueba el formulario de contacto
5. Haz clic en el botón para ver las interacciones

## Secciones del Sistema
1. **🏠 Home**: Página principal con información destacada de la compañía
2. **👨‍🚒 Bomberos**: Grid con listado de bomberos de la compañía
3. **ℹ️ Nosotros**: Información institucional de la compañía
4. **📝 Solicitud**: Formulario para postular como bombero voluntario
5. **📋 Citaciones**: Lista de citaciones y reuniones programadas

## Funcionalidades Implementadas
- **🧭 Navegación suave**: Menú con transiciones suaves entre secciones
- **🔐 Auth simulado**: Enlaces de "Iniciar sesión" y "Registrar usuario" (funcionalidad simulada)
- **👥 Tarjetas interactivas**: Grid de bomberos con efectos hover y modal informativo
- **📝 Formulario de solicitud**: Validación completa con campos específicos para bomberos
- **📋 Sistema de citaciones**: Información organizada de reuniones y eventos
- **📱 Responsive design**: Se adapta perfectamente a móviles y tablets
- **🎨 Efectos visuales**: Hover effects, transiciones y animaciones CSS

## Validaciones Implementadas
- ✅ Validación de edad mínima (18 años) para postulantes
- ✅ Validación básica de RUT chileno
- ✅ Validación de email con patrón regex
- ✅ Validación de campos obligatorios
- ✅ Feedback inmediato al usuario
